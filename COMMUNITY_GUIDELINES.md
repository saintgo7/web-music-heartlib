## Code of Conduct

### Expected Behavior

**We encourage:**
- Using welcoming and inclusive language
- Respecting differing viewpoints and experiences
- Accepting constructive criticism gracefully
- Focusing on what's best for the community
- Showing empathy towards other community members
- Giving credit where credit is due
- Helping newcomers feel welcome

### Unacceptable Behavior

**We do not tolerate:**
- Harassment, discrimination, or hate speech
- Personal attacks or insults
- Trolling or deliberately disruptive behavior
- Publishing others' private information
- Sexual attention or advances
- Spamming or excessive self-promotion
- Plagiarism or claiming others' work
- Deliberately spreading misinformation

### Scope

This Code of Conduct applies:
- In all project spaces (GitHub, Discord, forums)
- When representing the project publicly
- At project events and meetings
- In one-on-one interactions about the project

## Reporting Guidelines

### Reporting Bugs

When you find a bug:

1. **Check First**
   - Search existing issues
   - Try latest version
   - Review FAQ

2. **Gather Information**
   - OS and version
   - ABADA Studio version
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots/logs if applicable

3. **Submit Report**
   - Use bug report template
   - Be specific and detailed
   - One bug per issue
   - Tag appropriately

### Reporting Security Issues

**Do NOT** post security vulnerabilities publicly.

Instead:
1. Email: security@abada.kr
2. Include detailed description
3. Steps to reproduce (if applicable)
4. Potential impact
5. Suggested fix (optional)

We'll respond within 48 hours.

### Reporting Code of Conduct Violations

If you experience or witness unacceptable behavior:

1. **Document the Incident**
   - What happened
   - Where it occurred
   - When it happened
   - Who was involved

2. **Report Privately**
   - Email: conduct@abada.kr
   - Include documentation
   - We'll investigate promptly

3. **Confidentiality**
   - All reports handled confidentially
   - Reporter identity protected
   - Fair investigation process

## Moderation

### Our Approach

**Philosophy**: Light touch, clear rules, consistent enforcement

**Moderators**:
- Enforce Code of Conduct
- Resolve conflicts
- Remove inappropriate content
- Support healthy discussions
- Represent community values

### Enforcement Actions

**Progressive Response**:

1. **First Offense**: Private warning
2. **Second Offense**: Temporary ban (1-7 days)
3. **Third Offense**: Permanent ban

**Immediate Bans For**:
- Spam or malicious activity
- Harassment or abuse
- Sharing private information
- Impersonation
- Illegal content

### Appeals Process

If you disagree with moderation action:

1. Email: appeals@abada.kr
2. Explain your perspective
3. Provide context
4. Request review

Reviews handled within 7 days.

**Primary**: English (international communication)

We welcome translations:
- Documentation
- UI text
- Tutorial videos
- Community posts

Contact: translations@abada.kr

### Inclusive Language

**Use**:
- Gender-neutral terms
- Simple, clear language
- Define technical terms
- Provide examples
- Be patient with non-native speakers

**Avoid**:
- Jargon without explanation
- Assumptions about knowledge
- Cultural references that don't translate
