Show HN: ABADA Music Studio - One-click AI music generation (Windows/macOS/Linux)

### Submission Text

I built ABADA Music Studio to solve a problem I kept running into: AI music generation tools are powerful but incredibly difficult to set up.

**The Problem**

Tools like HeartMuLa can create amazing music, but require:
- Installing Python 3.10+, PyTorch, CUDA
- Managing 15+ dependencies
- Downloading 6GB+ models manually
- Debugging various platform-specific issues

Most non-technical users give up before creating a single song.

**My Solution**

A single installer that does everything:

1. Detects your OS and hardware (GPU support, RAM, disk)
2. Installs embedded Python environment
3. Downloads appropriate PyTorch version (CUDA/CPU/MPS)
4. Fetches AI models from HuggingFace
5. Sets up desktop shortcuts
6. Launches a local web UI

Total time: 15-30 minutes, zero user intervention.

**Technical Details**

Cross-platform challenges were interesting:
- Windows: NSIS installer with custom GPU detection (nvidia-smi parsing)
- macOS: Universal installer supporting Intel + Apple Silicon (MPS detection)
- Linux: Bash script supporting apt/dnf/pacman package managers

The installer embeds Python to avoid conflicts with system Python. Model caching is smart about resuming interrupted downloads.

Web UI is Gradio running locally, but deployed documentation site is on Cloudflare Pages for global reach.

**Architecture**

```
Installer (NSIS/Shell)

  ‚Üí PyTorch (CUDA/CPU/MPS)
  ‚Üí HeartMuLa-oss-3B
  ‚Üí Gradio UI
  ‚Üí Local browser (http://127.0.0.1:7860)
```

**Challenges**

1. Cross-platform GPU detection

4. Graceful degradation (GPU ‚Üí CPU)
5. Uninstaller cleanup
